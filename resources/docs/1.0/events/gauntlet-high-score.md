## Gauntlet High Score

---

This represents the high score achieved by players during the gauntlet.

### Structure

```json
{
    "type": "gauntlet_high_score",
    "round_id": "integer",
    "created_at": "string",
    "data": {
        "names": "string",
        "score": "integer",
        "highest_wave": "integer"
    }
}
```
